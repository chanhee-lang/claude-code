# BTW (By The Way)

코딩 중간에 떠오른 질문에 빠르게 답한다.
현재 작업 컨텍스트를 유지한 채로 답변 후 바로 작업으로 복귀한다.

## 규칙

- 답변은 짧고 핵심만. 길어지면 안 된다.
- 현재 열려 있는 파일이나 작업 중인 코드를 참고해서 답한다.
- 답변 후 "계속할게요" 같은 불필요한 말 없이 끝낸다.
- 질문이 현재 작업과 관련 있으면 코드 위치(파일:라인)를 함께 언급한다.

## 형식

```
> btw: [질문]

[한 문장~세 문장 이내의 답변]
```

## 예시

> btw: useEffect 의존성 배열에 함수 넣으면 매번 재실행돼?

함수가 렌더마다 새로 생성되면 그렇다. `useCallback`으로 감싸거나, 함수를 컴포넌트 밖으로 빼면 해결된다.

---

> btw: 이 API 호출 실패하면 어디서 잡아야 해?

현재 코드 기준으로 `fetchUser` (services/user.ts:23)에서 try-catch로 잡는 게 맞다. 컨트롤러까지 올리지 않는다.
